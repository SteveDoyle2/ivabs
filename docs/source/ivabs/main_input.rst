Main Input File
===============

The main input file uses the format YAML.

Overall layout of the main input file:

.. code-block:: yaml

    name: "composite_blade_design"
    version: "0.6"
    setting:
        # setting block
    design:
        # design block
    model:
        # model block
    analysis:
        # analysis block
    study:
        # dakota study block
